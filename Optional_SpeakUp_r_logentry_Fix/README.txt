【可选】SpeakUp — r_logentry 语法兜底修复

使用方式：
  将此整个文件夹 “Optional_SpeakUp_r_logentry_Fix” 复制到
  RimWorld/Mods/  下（与 DD-oher zh 同级），
  然后在游戏中勾选该 mod，排在 SpeakUp 之后即可。

作用：
  为 SpeakUp 的互动 def 增加一条无条件的 r_logentry→[good_opinion] 规则，
  在 Bubbles 等未传入 opinion 时仍能解析，避免 "Grammar unresolvable. Root 'r_logentry'"。
  对所有语言生效（英文/中文等）。

若你已使用 DD-oher zh 汉化并在中文下游玩，汉化里已批量加了兜底，可不启用本补丁。
